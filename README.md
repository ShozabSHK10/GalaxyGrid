## Quick Start

1. **Clone and install**
   ```bash
   git clone https://github.com/YOUR_USERNAME/GalaxyGrid.git
   cd GalaxyGrid
   npm install
   ```
   
2. **Setup Firebase**
   - get a free NASA API key from [NASA API KEY](https://api.nasa.gov/)
   - Create a .env file in the project root
   ```bash
   VITE-NASA-API-KEY = YOUR_API_KEY_HERE
   ```

3. **Run**
   ```bash
   npm start
   ```
