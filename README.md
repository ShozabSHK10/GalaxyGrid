## Quick Start

1. **Clone and install**
   ```bash
   git clone https://github.com/ShozabSHK10/GalaxyGrid.git
   cd GalaxyGrid
   npm install
   ```
   
2. **Setup the API key**
   - get a free NASA API key from [NASA API KEY](https://api.nasa.gov/)
   - Create a .env file in the project root
   ```bash
   VITE_NASA_API_KEY = YOUR_API_KEY_HERE
   ```

3. **Run**
   ```bash
   npm start
   ```
